PSYCHE AR/XR For Public Engagement - Kristen, Neil, Wilson, Thuy, Angie!

Instructions for local testing:

1. Make sure you have the SDK from Variant Launch in index.hmtl! (do not push it)
1. npm install
1. npm run serve-public

Instructions for deployed site updates:

Assuming new pushes/commits added,

1. npm run build
2. Copy /build folder CONTENTS (do not copy the actual /build folder)
3. Paste into Psyche AR-XR repo
4. Commit new build on the deployed repo
5. Enjoy!
